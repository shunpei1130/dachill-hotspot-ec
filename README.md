# DACHILL Hotspot EC

漫画の世界を探索しながら、商品やブランドを知る「漫画型ホットスポットEC」のプロトタイプです。

## User test

https://dachill-hotspot.shunpeihasegawa.chatgpt.site

## Features

- モバイルファーストの1ページ構成
- 漫画イラスト全面表示
- 画像比率に追従する11個の透明ホットスポット
- オブジェクトごとに異なる漫画風モーダル
- 商品モーダル内だけに購入導線を配置
- FAQアコーディオン・お問い合わせフォーム
- PCホバー／スマートフォンタップ／キーボード操作対応
- Shopify／Stripeへ差し替え可能な購入リンク構成

## Development

```bash
npm install
npm run dev
```

## Status

価格、在庫、会社概要、法務情報、配送条件、購入リンクはプロトタイプ用の仮情報です。
